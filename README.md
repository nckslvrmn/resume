# resume

My codified resume. Uses Github Actions to publish the web version to GitHub Pages and build `resume.pdf` from the same JSON source.

The site also has a PDF button that opens the browser print dialog with a light, paper-focused stylesheet. In the deployed Pages artifact, `resume.pdf` is generated automatically during publish.

Original concept and schema inspired by [jsonresume](https://jsonresume.org/). Original theme inspired by [kwan](https://github.com/icoloma/jsonresume-theme-kwan)
